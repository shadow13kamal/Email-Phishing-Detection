About This Project

Phishing Email Detection System — BCA Cyber Security Mini Project

Project Overview

Phishing is a cyber-attack where attackers send fraudulent emails that appear to come from legitimate sources to steal sensitive information such as passwords, credit card numbers, and personal data. This project builds a Machine Learning-based web application that automatically detects whether an email is a phishing attempt or safe.

The system uses Natural Language Processing (NLP) techniques to preprocess email text, TF-IDF for feature extraction, and three classification algorithms — Logistic Regression, Naive Bayes, and Random Forest — to identify phishing patterns. The best-performing model is automatically selected and deployed.

How the System Works

  1. Data Collection: A labeled dataset of phishing and legitimate emails is loaded from a CSV file.
  2. Text Preprocessing: Email text is lowercased, punctuation is removed, and stop words are filtered out using NLTK tokenization.
  3. Feature Extraction: The TF-IDF vectorizer converts preprocessed text into a numeric feature matrix with unigram and bigram features.
  4. Model Training: Three classifiers are trained on 80% of the data. They are evaluated on the remaining 20% using accuracy, precision, recall, and F1 score.
  5. Model Selection: The model with the highest F1 score is automatically selected and saved using Joblib.
  6. Web Interface: A Flask web application lets users paste emails and receive instant predictions with confidence scores, risk levels, and highlighted indicators.

Phishing Indicators Detected

Suspicious URLs

  • URLs with suspicious TLDs (.tk, .ml, .cf, .ga, .gq)
  • URLs impersonating legitimate brands
  • Shortened URLs (bit.ly, tinyurl)
  • URLs with misspelled domain names

Suspicious Keywords

  • Urgency phrases ("act now", "immediate action")
  • Account threats ("account suspended", "account locked")
  • Financial lures ("lottery", "you have won", "gift card")
  • Credential requests ("verify your account", "enter password")

Academic Context

This project is submitted as part of the Bachelor of Computer Applications (BCA) Cyber Security curriculum. It demonstrates the practical application of Machine Learning and Natural Language Processing to solve real-world cybersecurity problems.
